Projects

Please check out my projects below. Feel free to test them out by clicking through either the picture or the project title. These projects were completed during 2018 and 2019.

berm

A full-stack PERN (Postgres, Express & Sequelize, React, Node) mountain-biking app. Create an account and track your bikes and bike components, keep track of trails you've ridden and wish to ride, and log track times.

Specialized Recipe Search

A JavaScript web application that utilizes the Edamam recipe search API. Enter a food, select diet types, health options, and food/recipe exclusions and receive recipes based on your criteria.

Travall

Travall is a collaborative travel-planning app that helps you stay organized with others about your upcoming travel plans. I built out the server for this app in Node.js, setting up relational database tables with Express functions. I also worked extensively on the Angular 7 front-end, applying TypeScript features. I was Git master for this group project, and both sides of Travall can be found pinned on my Github.

JavaScript Fundies

A static webpage describing vanilla JavaScript arrays, functions, and objects. This was my first project at Eleven Fifty Academy. The inspiration for the style of this page is inspired by the VS Code color scheme Monokai Pro, the color scheme I most enjoy using in VS Code.
